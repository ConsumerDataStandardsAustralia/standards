V1.32.0 Release Notes

Release notes for version v1.32.0 of the CDR Standards.

Changes Made

Change Requests

This release addresses the following minor defects raised on Standards Staging:

  • Standards Staging #410 - Clarify RateString description and examples
  • Standards Staging #411 - Fix typos
  • Standards Staging #413 - Merge Change Log and Archives sections
  • Standards Staging #417 - Non-existent property "amount" listed as required
  • Standards Staging #418 - Add a link to CX Guidelines to the left menu pane of the Standards
  • Standards Staging #419 - Field name styling
  • Standards Staging #426 - Error in Telco OAS components
  • Standards Staging #437 - Remove Known Issues
  • Standards Staging #438 - Improve code sample section styling

This release addresses the following change requests raised on Standards Maintenance:

  • Standards Maintenance #641 - Update CDS documentation to clarify expected rate value 'sign' (+/-) for each RateType
  • Standards Maintenance #647 - Maintenance Iteration 20 Holistic Feedback
  • Standards Maintenance #648 - Adopt BCP 195 for TLS ciphers
  • Standards Maintenance #652 - Specify units of currency to be used for the AmountString field type
  • Standards Maintenance #653 - EnergyPlanTariffPeriod - cater for plans with no dailySupplyCharge

Decision Proposals

This release addresses the following Decision Proposals published on Standards:

  • Decision Proposal #353 - Maintenance Iteration 20
