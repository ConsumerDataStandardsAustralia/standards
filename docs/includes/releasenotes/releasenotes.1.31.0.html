V1.31.0 Release Notes

Release notes for version v1.31.0 of the CDR Standards.

Changes Made

Change Requests

This release addresses the following minor defects raised on Standards Staging:

  • Standards Staging #377 - Review FDO table
  • Standards Staging #394 - Fix typo 'registeration'
  • Standards Staging #395 - Enhancements to Banking documentation
  • Standards Staging #396 - Improve clarity of the PerformanceMetricsV3 structure
  • Standards Staging #400 - Enable generic links to schemas
  • Standards Staging #402 - Update Consumer Data Right link

This release addresses the following change requests raised on Standards Maintenance:

  • Standards Maintenance #415 - Disambiguation of the claims for a response from the introspection endpoint
  • Standards Maintenance #615 - Plan Obligation Milestones for 2025
  • Standards Maintenance #633 - Collection Consents - Authorisation Amendment
  • Standards Maintenance #634 - Potential error in Telco specification
  • Standards Maintenance #640 - Retirement date for Get Generic Plan Detail v2 and Get Energy Account Detail v3
  • Standards Maintenance #643 - Update TLS cipher suite requirements to address DHEat Attacks and Raccoon Attack vulnerabilities

Decision Proposals

This release addresses the following Decision Proposals published on Standards:

  • Decision Proposal #347 - Maintenance Iteration 19
