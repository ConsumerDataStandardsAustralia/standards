V1.22.0 Release Notes

Release notes for version v1.22.0 of the CDR Standards.

Changes Made

Change Requests

This release addresses the following minor defects raised on Standards Staging:

  • Fix for a missing font in styles
  • Broken links in Register API section

This release addresses the following change requests raised on Standards Maintenance:

  • Iteration 13 Holistic Feedback
  • Specify if an Account is a joint account in the API response
  • Update x-v header to be mandatory for Register APIs
  • Update Register and DCR Swagger specs to use Common Field Types

Decision Proposals

This release addresses the following Decision Proposals published on Standards:

  • Decision 272 - Maintenance Iteration 13
  • Decision Proposal 275 - Holistic Feedback on Telco Standards - Note that this is not a formal decision of the chair. The Telco standards remain in draft form
